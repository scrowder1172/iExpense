# Project 07: iExpense

## Hacking With Swift project 07

This is Project 07 in Paul Hudson's 100 Days of SwiftUI course. This project creates an iOS app that the allows the user to record expenses. The expenses are grouped into different categories.

## Course Link

Check out Paul's course for more on SwiftUI: https://www.hackingwithswift.com/100/swiftui

## How to install

1. fork the project or download the .zip
2. open project in XCode v15 or later (earlier versions may or may not compile correctly)
3. run in simulator
